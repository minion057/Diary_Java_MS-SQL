#실행 순서
1. 첫 실행 화면에서 로그인을 해야 넘어갈 수 있음
(이때 로그인하는 부분에서 회원가입 탭과 아이디, 암호 찾기 탭을 만들어 로그인하게 했다)
2. 가입할 때 작성한 아이디'Diary 라는 화면이 뜨고, 현재 날짜가 보이는 달력, 클릭된 날짜 스케줄러, 현재시간이 보이는 화면이 뜬다.
3. 스케줄러 추가, 수정, 삭제, 일기장으로 넘어가기, 환경설정으로 가기 등등 여러 화면으로 넘어갈 수 있다.

#기능 설명
1. 로그인, 회원가입, 정보찾기(아이디, 암호), 회원탈퇴를 DB와 연동하여 구현
2. 로그인 한 후에는 접속한 user의 아이디'Diary라는 글자가 보이게 구현
3. 달력은 현재 날짜를 기준으로 보이게 함(오늘 혹은 사용자가 클릭한 날은 보라색 바탕, 기념일이 있는 날은 청록색 글씨, 주말은 빨간 바탕으로 구현)
4. 현재 보이는 달력 기준으로 '<<','<','>','>>' 버튼으로 작년, 저번달, 다음달, 내년의 달력을 보여준다.
5. Today 버튼으로 오늘 날짜가 있는 달력으로 돌아오게 구현
6. 콤보 박스를 이용해 여러 년도를 이동할 수 있음
7. 스케줄은 할 일과 기념일로 구분함(스케줄 추가시 기본으로 할 일을 선택하게 함, 버튼도 글자가 들어와야 활성화 됨)
8. 로그인 한 화면이 켜지자마자 Thread를 활용해 실시간 시간을 보여줌
9. 스케줄 아래 'Write a diary' 버튼을 통해 일기장으로 넘어감
